# Verilog HDL for Digital Design

This repository contains Verilog HDL implementations of fundamental digital circuits and systems. It is intended for beginners, ECE students, and anyone interested in learning Digital Logic Design through practical Verilog coding examples.

## What You'll Find

This repository covers a wide range of Digital Logic Design concepts, starting from basic logic gates and gradually progressing toward more advanced combinational and sequential circuits.

Topics include:

* Basic Logic Gates
* Adders and Subtractors
* Encoders and Decoders
* Multiplexers and Demultiplexers
* Comparators
* Latches and Flip-Flops
* Registers
* Counters
* Finite State Machine (FSM) Designs
* Seven-Segment Display Designs
* Digital Logic Mini Projects

More designs and projects will be added as I continue learning and exploring Digital Design.

## Repository Features

Each module may include:

* Verilog HDL Source Code
* Testbench Files
* Simulation Results and Waveforms
* Truth Tables
* Logic Diagrams and Block Diagrams
* Notes and Explanations for Better Understanding

The goal is not only to provide working Verilog code but also to explain the underlying concepts behind each design.

## Software Used

All designs in this repository are developed, simulated, and verified using:

* Xilinx Vivado Design Suite 2016.4

A beginner-friendly guide for downloading, installing, and setting up Vivado will also be provided.

## Who Is This Repository For?

* ECE Students
* FPGA Beginners
* VLSI Enthusiasts
* Digital Design Learners
* Anyone Starting Their Verilog HDL Journey

## Goal

The objective of this repository is to build a structured collection of Verilog HDL implementations, projects, and learning resources that help students develop a strong foundation in Digital Logic Design and Hardware Description Languages (HDLs).

Whether you are learning your first logic gate or building complex digital systems, this repository aims to serve as a practical reference throughout your learning journey.
